# Currency converter

A simple python currency converter using real_time exchange rates from an API.

## Features
-Convert between different currencies(USD,EUR,IRR, etc.)
-Real-time exchange rates
-Easy-to-use terminal interface

## Requirements
-Python 3
-requests library
